# 과제 주제 연결 문단

아래 문단을 개발계획서·완료보고서 HTML에 복사해 넣으세요.

---

## 1. 프로젝트 개요 — 배경 (계획서·보고서 공통)

> 본 프로젝트는 **전국 각 지역을 순회하는 JLPT 오프라인 설명회·박람회** 참가 수험생을 위한 단어 학습 서비스입니다. 서울, 부산, 대구, 광주, 대전 등 지역별로 개최되는 JLPT 행사에서는 수험생들이 현장에서 즉시 학습할 수 있는 도구가 필요하지만, 행사장 Wi-Fi 환경이 불안정한 경우가 많습니다. 이에 본 앱은 **오프라인 우선 설계**로 인터넷 연결 없이도 JLPT N5~N1 단어를 학습·테스트할 수 있도록 구현하였습니다.

---

## 2. 프로젝트 목적 (계획서용)

> - 지역별 찾아가는 JLPT 오프라인 행사 참가자에게 **현장 즉시 학습** 환경 제공  
> - React, JavaScript, Git 등 학습 기술을 활용한 **작동 가능한 웹/모바일 프로젝트** 완성  
> - 기획 → 구현 → 제출까지 **개인 단위로 end-to-end** 개발 경험 축적  
> - 행사 일정 안내와 단어 학습을 하나의 앱에 통합하여 **실용적 서비스** 구현

---

## 3. 타겟 사용자 (계획서용)

> | 사용자 | 니즈 |
> |--------|------|
> | JLPT 오프라인 행사 참가자 | 행사장에서 바로 단어 복습 |
> | 지역 수험생 (수도권 외) | 원거리 이동 전·후 학습 연속성 유지 |
> | 오프라인 환경 학습자 | Wi-Fi 없이 플래시카드·테스트 이용 |

---

## 4. 핵심 기능 — 행사 안내 (보고서용, 신규 추가 섹션)

> ### 지역별 오프라인 행사 안내 기능 (신규)
>
> 과제 주제인 **「지역별 찾아가는 오프라인 행사」**에 맞춰, 전국 5개 지역(서울·부산·대구·광주·대전) JLPT 행사 일정을 안내하는 화면을 추가하였습니다.
>
> - **홈 화면**: 다가오는 행사 2건 미리보기
> - **행사 안내 화면**: 지역, 일정, 장소, 대상 레벨, 행사 설명
> - **행사 맞춤 학습**: 행사별 추천 JLPT 레벨로 바로 플래시카드 학습 시작
>
> 이를 통해 앱이 단순 학습 도구를 넘어, **지역 오프라인 행사와 연계된 통합 서비스**로 확장되었습니다.
